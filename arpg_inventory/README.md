# ARPG Inventory & Trade Manager

Автор: Грандов Максим
Вариант: ARPG Trade Manager (ООП, MVC)
Дата сдачи: 21.06.2026

## Описание программы
Консольное приложение для учета внутриигровых предметов и валюты. Позволяет добавлять вещи в тайник, автоматически сохранять их в базу данных, фильтровать по категориям для быстрого поиска нужных характеристик и отслеживать историю лута с помощью структуры данных Стек.

## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше

## Как запустить
1. Откройте терминал в папке проекта.
2. Запустите главный файл командой:
```bash
python main.py
